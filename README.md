# Traffic Light Controller (VHDL)

## Project Overview

This project implements a Traffic Light Controller using VHDL. The system controls traffic signals for North-South and East-West directions and includes a pedestrian crossing feature.

## Features

* VHDL-based digital design
* Finite State Machine (FSM) implementation
* North-South and East-West traffic control
* Pedestrian crossing button support

## Tools Used

* VHDL
* FPGA / Digital Design
* Simulation tools (ModelSim / Vivado / etc.)

## Project Structure

* `traffic_light_controller.vhd` : Main VHDL design
* `testbench.vhd` : Simulation testbench

## How It Works

The system uses a finite state machine to control the sequence of traffic lights. The controller changes the states based on timing and pedestrian input signals.
